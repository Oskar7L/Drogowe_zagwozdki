# Data Engineer

Ten folder przeznaczony jest dla Data Engineerów pracujących nad projektem Drogowe Zagwozdki.

## Zakres odpowiedzialności
- Budowa i utrzymanie pipeline'ów danych
- ETL / ELT – pobieranie, transformacja i ładowanie danych
- Zarządzanie bazami danych i hurtowniami danych
- Zapewnienie jakości i dostępności danych dla innych ról

## Struktura
```
data_engineer/
├── pipelines/      # Skrypty ETL/ELT
├── schemas/        # Definicje schematów baz danych
└── docs/           # Dokumentacja architektury danych
```
