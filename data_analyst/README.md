# Data Analyst

Ten folder przeznaczony jest dla Data Analityków pracujących nad projektem Drogowe Zagwozdki.

## Zakres odpowiedzialności
- Analiza statystyczna i eksploracja danych
- Tworzenie raportów i wizualizacji
- Odpowiadanie na pytania biznesowe za pomocą danych
- Przygotowanie danych do dashboardów

## Struktura
```
data_analyst/
├── reports/        # Raporty i prezentacje wyników
├── queries/        # Zapytania SQL i skrypty analityczne
├── visualizations/ # Wykresy i wizualizacje statyczne
└── docs/           # Dokumentacja analiz
```
