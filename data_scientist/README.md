# Data Scientist

Ten folder przeznaczony jest dla Data Scientistów pracujących nad projektem Drogowe Zagwozdki.

## Zakres odpowiedzialności
- Eksploracja i analiza danych (EDA)
- Budowa i ewaluacja modeli machine learning
- Eksperymenty predykcyjne i klasyfikacyjne
- Przygotowanie wyników do wdrożenia

## Struktura
```
data_scientist/
├── notebooks/      # Jupyter Notebooks z analizami i modelami
├── models/         # Zapisane modele (pickle, joblib, ONNX itp.)
├── experiments/    # Logi i wyniki eksperymentów
└── docs/           # Dokumentacja metodologii
```
